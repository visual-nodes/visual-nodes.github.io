visual-nodes

A new framework for generating VPLs (visual programming languages)

Experimental

visual-nodes is an experimental project.
The api is not finished and could change anytime!

Focus on HomeAutomation

visual-nodes could be used for anything, but we have to start somewhere.
We decided to start with home-automation, because we needed it personally.

Powered by NodeJS

visual-nodes utilizes NodeJS and the NPM ecosystem to modularize the software.
Currently an internal NPM-Registry is used for management, but this could change in the future.
